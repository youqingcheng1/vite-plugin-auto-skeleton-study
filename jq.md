### jq
```
$('body') =>返回jq获取的节点
$('body').eq(0) =>返回第一个索引
$('body').addClass('sk') => 添加class
$('body').attr('data-root','text') => 添加属性 data-root=text
$('body').attr('data-root') => 返回属性 data-root 的值
$('body').prop('outerHTML') => 输出内容一个html 字符串
$('body').children() => 返回子节点
$('body').empty() => 清空节点
$('body').text() => 返回文案
$('body').insertAfter() => 在body插入节点
$('body').move => 删除节点
$('body').contents() => 返回body下的所有节点
```
